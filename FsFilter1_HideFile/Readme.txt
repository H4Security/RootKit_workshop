A simple Windows file system filter driver create mainly for demonstrating some mini-filter (filter driver) functionality and how they can affect the data flow from user space to kernel space. The filter will block any request (read or write) to any file have the name "HideMe.txt" So before you start the service, you should create "HideMe.txt" file. After that, start installing the service by right click on the FsFilter1_HideFile.inf and install it, then run service as privileged account "sc start FsFilter1_HideFile"
